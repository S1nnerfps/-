## 二、产品定位演进

### 初始版本（已废弃）
- 赛前静态战术板工具
- 用户手动在地图上标记烟/闪/路线
- AI 评估 + 优化静态战术

### 当前版本（v2 — 实时战术智能系统）
> 灵感来自 DMA，但**不接入游戏进程、不读内存、不违反反作弊**。
> 运行在第二屏幕（平板/笔记本），纯靠用户手动输入 + AI 推理。

- **数据入口**：玩家死亡视角时在第二屏幕快速标记对手信息
- **AI 分析**：回合间隙快速分析 + 暂停/中场深度分析
- **数据分区**：己方车队数据（持久化）+ 对手数据（每局临时）

## 三、核心架构决策

### 3.1 三个时间层

| 时间层 | 触发时机 | 响应速度 | 深度 | 输出 |
|--------|----------|----------|------|------|
| 🟢 实时输入 | 死亡视角期间 | 即时保存 | 浅（仅记录） | 后台累积数据 |
| 🟡 快速分析 | 回合间隙（~15s） | 3-5s | 中 | 一句建议 + 置信度标注 |
| 🔴 深度分析 | 暂停（30s）/中场 | 15-20s | 深 | 对手画像 + 战术调整方案 |

### 3.2 数据分区

#### 数据集 A：己方车队（持久化，跨对局积累 — localStorage）
| 维度 | 积累内容 | AI 用途 |
|------|----------|--------|
| 交火习惯 | 队员各位置击杀/被击杀 | 推荐符合习惯的站位 |
| 优势选位 | 队员常驻且胜率高的位置 | 战术生成优先分配 |
| 高成功率战术 | 车队历史战术胜率 | 评估报告引用 |
| 投掷物执行力 | 队员烟/闪成功率 | 分配投掷物任务 |
| 沟通偏好 | 队员报点方式/简称 | 个性化战术说明 |

#### 数据集 B：当前对手（每局新建，赛后丢弃 — 内存变量）
| 维度 | 观察内容 | 输入时机 | AI 用途 |
|------|----------|----------|--------|
| 防守站位倾向 | CT 各位置出现频率 | 死亡视角 | 预测防守布局 |
| 进攻路线偏好 | T 方主攻方向统计 | 死亡视角 | 预测进攻方向 |
| 投掷物习惯 | 对手常用烟/闪点 | 死亡视角 | 反制建议 |
| 经济行为 | Eco/Rush/Full Buy 节奏 | 回合间隙 | 预测装备等级 |
| 队内核心识别 | 首杀最多者/指挥 | 暂停分析 | 压制核心 |

### 3.3 三种分析模式

| 模式 | 触发 | 响应 | 置信度 | 输出 |
|------|------|------|--------|------|
| 快速分析 | 回合间隙 | 3-5s | ~60-70% | 一句话建议 |
| 深度分析 | 暂停/中场 | 15-20s | ~85%+ | 对手画像 + 战术方案 |

## 四、页面结构

```
🏠 首页
 ├─ 🆕 开赛（新对局）
 │    ├─ 选择地图
 │    ├─ 输入对手队名（可选）
 │    └─ 加载己方车队历史数据
 │         └─ 🎮 比赛仪表盘（核心页面）
 │               ├─ 地图缩略图（实时标记汇总）
 │               ├─ 当前比分 + 经济条
 │               ├─ 死亡输入面板（条件显示）
 │               ├─ 快速分析弹窗（回合间隙弹出）
 │               └─ 深度分析面板（暂停弹出）
 ├─ 📐 战术编辑器
 │    └─ 赛前预设 / 深度分析结果修改
 └─ 📊 车队数据中心
      └─ 历史对局统计 + 队员数据趋势
```

## 六、AI 评估维度（5 维）

| 维度 | 评分逻辑 | 依赖标记 |
|------|----------|----------|
| 控图覆盖率 | 控图区域是否覆盖进攻/包夹起点 | 控图 + 进攻 + 包夹 |
| 投掷物效率 | 烟雾是否封住防守防线，闪光是否覆盖清点路径 | 烟/闪 + 防守 + 反清 |
| 影响力重叠度 | 队员影响力区域是否有死角/过度重叠 | 影响力 + 站位 |
| 时间窗口可行性 | 基于标记间距离估算执行秒数 | 进攻 + 包夹 |
| 风险等级 | 防守防线覆盖密度 vs 进攻方向数量 | 防守 + 进攻 + 反清 |

## 七、关键页面设计

### 7.1 首页 — 地图选择
- 7 张地图卡片（Mirage/Inferno/Dust2/Nuke/Ancient/Anubis/Vertigo）
- 每张显示地图缩略图 + 名称
- 深色背景（游戏感），卡片圆角，hover 高亮

### 7.2 战术编辑器
- 顶栏：战术名输入框 + 风格下拉（激进/保守/均衡）+ AI分析按钮 + AI优化按钮
- 左侧：11 工具全展示工具栏
- 中央：Canvas 地图实时绘制区
- 底栏：当前工具 + 操作提示

### 7.3 AI 评估弹窗
- 五维雷达图（Canvas 绘制）
- 综合评分 /100
- 优势列表 + 风险列表
- 按钮：[手动调整] [让AI优化]

### 7.4 AI 优化对比弹窗
- 左：原始方案缩略图 + 评分
- 右：AI 优化方案缩略图 + 评分
- 变更说明列表
- 按钮：[保留原方案] [采用AI优化]

### 7.5 比赛仪表盘（新增核心页面）
- 地图缩略图（实时标记汇总显示）
- 比分栏 + 双方经济趋势条
- 死亡输入面板（条件显示 — 仅在死亡视角时出现）

### 7.6 死亡输入面板
- 对手位置：地图点击标记 + 快捷按钮（A点/B点/中路/A小/B小/窗口/长廊/匪家/警家）
- 对手武器：快捷按钮（AK47/M4/AWP/SMG/霰弹）
- 对手行为：快捷按钮（前压/守点/绕后/双架）
- [确认记录] / [跳过] 按钮

### 7.7 快速分析弹窗（回合间隙 ~15s）
- 对手倾向 + 置信度 %
- 一句战术建议
- 按钮：[忽略] [采用建议]

### 7.8 深度分析面板（暂停/中场）
- 对手画像（主攻方向柱状图、防守站位分布、核心选手标识、经济趋势）
- 针对性战术建议（多条）
- 按钮：[导出到编辑器] [语音播报]
